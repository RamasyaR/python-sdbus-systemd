Systemd binds for python-sdbus
==============================

This package is a wrapper based python-sdbus for the D-Bus interface
of `Systemd <https://www.freedesktop.org/wiki/Software/systemd/>`_.

.. note::
    See `upstream documentation for Systemd D-Bus API
    <https://www.freedesktop.org/software/systemd/man/org.freedesktop.systemd1.html>`_
    for more detailed documentation.
